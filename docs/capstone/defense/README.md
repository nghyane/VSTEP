# Defense Guide — VSTEP Platform

Tài liệu này dùng để học ý chính trước buổi bảo vệ. Không đọc như báo cáo dài; hãy đọc theo thứ tự dưới đây.

## Luồng đọc nhanh

1. `01-dan-y-thuyet-trinh.md` — flow thuyết trình và trọng tâm demo.
2. `02-he-thong-chuc-nang.md` — toàn bộ chức năng chính của sản phẩm.
3. `03-assessment-engine.md` — cách giải thích hệ thống chấm Writing/Speaking.
4. `04-validation-khong-co-vstep-chuan.md` — cách trả lời khi không có bộ dữ liệu VSTEP chuẩn.
5. `05-kich-ban-demo-web.md` — kịch bản demo web.
6. `06-gioi-han-va-cau-hoi.md` — giới hạn và Q&A ngắn để học thuộc ý.

## Thông điệp trung tâm

> Đây là nền tảng luyện thi VSTEP thích ứng, bao gồm luyện tập 4 kỹ năng, luyện Writing/Speaking có feedback, từ vựng, mock test, learning path và progress dashboard. Assessment engine là module lõi để hỗ trợ phản hồi minh bạch, không thay thế giám khảo hoặc điểm thi chính thức.

## Ba điều cần giữ nhất quán

- Không claim “chấm chuẩn như giám khảo VSTEP”.
- Không gọi kết quả kiểm thử nội bộ là độ chính xác chính thức.
- Luôn nhấn mạnh: có căn cứ chấm điểm, bám tiêu chí đánh giá, có kiểm thử tình huống rủi ro và hỗ trợ học tập.

## Thuật ngữ nên dùng khi bảo vệ

| Không nên nói | Nên nói |
|---|---|
| golden probes | bộ mẫu kiểm thử nội bộ |
| guardrails | kiểm thử tình huống rủi ro / cơ chế giới hạn điểm |
| practice score | điểm luyện tập |
| benchmark | bộ dữ liệu đánh giá / bộ mẫu đối chiếu |
| official accuracy | độ chính xác chính thức so với giám khảo |
| rubric alignment | bám theo tiêu chí đánh giá |
| reference signals | tín hiệu chấm điểm có nguồn tham chiếu |
| pipeline | quy trình xử lý/chấm điểm |
| skill gaps | điểm yếu theo kỹ năng |
| spaced repetition | ôn tập ngắt quãng |
