# Practice UX Decisions

## Transcript/hỗ trợ miễn phí trong luyện tập

Luyện tập (practice) hiện transcript, dịch, keywords miễn phí. Không trừ xu.
- Lý do: khuyến khích user dùng hỗ trợ để học tốt hơn, không tạo rào cản
- Xu dùng cho: thi thử (mở đề), khóa học premium
- Backend support level API giữ nguyên cho thi thử nếu cần

## Listening layout

Audio player nằm trên, câu hỏi nằm dưới (scroll cùng nhau).
Subtitle sticky khi CC bật — user vừa đọc vừa trả lời.
Footer chỉ có nav pills + submit.
